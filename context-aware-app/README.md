# 🚀 Nexus (Ctrl-Ctrl): 你的上下文感知效率利器

![Nexus Logo](https://via.placeholder.com/150) <!-- 建议替换为你的应用图标 -->

**Nexus** 是一款专为 Windows 打造的轻量级生产力工具。它能够“感知”你选中的文本内容，并在你双击 `Ctrl` 时，立即为你推荐最合适的操作：无论是翻译、搜索、科学文献检索，还是 AI 深度处理，只需一键。

---

## ✨ 核心特性

- **⚡ 闪电触发**：全局双击 `Ctrl` 键，瞬间唤醒，从不打断你的心流。
- **🧠 场景感知**：智能识别选中的内容（网址、DOI、文件路径、普通文本），只显示你需要的操作。
- **🤖 AI 赋能**：内置强大的 AI 接口，支持沉浸式全文翻译、智能总结及文本润色。
- **🛠️ 高度定制**：支持自定义正则表达式匹配逻辑，打造属于你自己的快捷指令库。
- **🦀 极致轻量**：基于 Rust & Tauri 开发，内存占用极低，运行流畅。

## 📥 安装与使用

1. **下载**: 前往 [Releases](https://github.com/你的用户名/仓库名/releases) 下载最新的 `Nexus-Setup.exe`。
2. **启动**: 安装并运行程序，它会静默在系统托盘。
3. **快捷键**: 
   - **双击 Ctrl**: 针对选中文本弹出菜单。
   - **右键图标**: 进入设置页面，自定义你的 AI Key 和匹配规则。

## 🛡️ 安全与隐私

针对目前仅发布二进制版本，我们明白安全的重要性：
- **数据本地化**: 除非你使用 AI 功能，否则所有文本处理和匹配逻辑均在本地完成。
- **VirusTotal 报告**: [点击查看最新版本的安全扫描记录](https://www.virustotal.com/gui/home/upload) (建议上传你的 exe 后获取链接)。
